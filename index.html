ReVanced Helper

what to do before running the command?

(thanks to paget96projects for this article, i took most of this part from there)

1. get JDK 17

if you do not have jdk 17 you can download if from here (choose your OS and architecture)
to verify run "java --version" (without the quotation mark)

2. Download the files

you need these files:

  • revanced cli (file name is going to be something like revanced-cli-X.X.X-all.jar)
  • revanced patches (download the jar)
  • revanced integrations (download the apk)
  • youtube apk version 17.22.36

3. renaming files (so it works with the command)

(X is a random number)

  • rename "revanced-cli-X.X.X-all.jar" to "revanced-cli.jar"
  • rename "revanced-patches-X.X.X.jar" to "revanced-patches.jar"
  • rename "app-release-unsigned.apk" to "integrations.apk"
  • rename the youtube apk (it has a long name) to "youtube.apk"

4. get microG

you can download it from the the official website (Services Core)
another way is to download F-Droid, add IzzyOnDroid F-Droid Repository, and download Vanced microG

5. install youtube

disable youtube from your android device and install youtube.apk (the one you downloaded)

6. setting up

enable USB Debugging (you can use this guide),
run "adb devices" and get the device ID (it is before "device")
